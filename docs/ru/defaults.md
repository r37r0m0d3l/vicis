# Defaults - По Умолчанию

---

◀ [Конфигурация](/ru/configuration.md)

Если свойство имеет неопределенное значение, оно будет заменено значением из конфигурации

```js
const { defaults } = require("vicis");
```

```js
import { defaults } from "vicis";
```
